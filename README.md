## 🚀 Overview

**Bank Account Manager** allows users to create and manage bank accounts, perform transactions, and track balances efficiently.  
The project focuses on correctness, simplicity, and clean financial logic.

## ✨ Features

- 🏦 Create and manage multiple bank accounts  
- 💰 Deposit and withdraw funds  
- 🔄 Transfer money between accounts  
- 📊 Real-time balance updates  
- 🧾 Transaction history tracking  
- 🔐 Validation to prevent invalid transactions
